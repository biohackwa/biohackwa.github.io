Schedule

We will start with a short history of the Biohacking field. You will learn GIT and GITHUB as version control and documentation software. Next, you will learn how to design objects in 2D and 3D and transform them into real objects with a lasercutter or 3D printer. A brief intro to electronics will prepare you for next session. During the lab days as a demo we will make a flow hood as a team.

Agenda

  • Device: Sterile floow hood
  • Git version control
    • Git client & UI
    • Github
  • Github Pages and documentation
  • Sterile hood design
  • 3D design in SketchUp
  • Introduction to electronics
  • Breadboarding - Introduction
    • Basic LED circuit
    • LED driver
